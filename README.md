# O que é um QA e qual seu papel em um time ? 

De maneira abrangente, QA (Quality Assurance) é uma função dentro de um time de desenvolvimento de software que atua na prevenção de bugs e assegura a qualidade do produto, desde a concepção da ideia até o lançamento.

No entanto, o QA vai muito além disso, tornando-se uma cultura intrínseca ao ciclo de vida do software. Profissionais de QA são responsáveis por analisar todo o ciclo de desenvolvimento de um produto digital. Sua função envolve assegurar o correto funcionamento dos sistemas, revisar os prerrequisitos do software e identificar possíveis falhas, colaborando diretamente com o time de desenvolvimento.
